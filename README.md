# job-portal-mastermind

A website where people can sign up and create job with proper job details and then they should be able to edit also . All active jobs should be displayed on home page and it should be able to integrate with any other rest API so that candidates who are applying to these jobs can be sent to employers

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/job-portal-mastermind.git
cd job-portal-mastermind
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
